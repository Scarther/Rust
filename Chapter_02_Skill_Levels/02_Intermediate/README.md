# Intermediate Level Projects (I01-I15)

## Overview

Intermediate projects build on Basic foundations with networking, concurrency, and more complex system interactions.

## Prerequisites

- Completed Basic Level (B01-B15)
- Understanding of ownership and borrowing
- Familiarity with error handling

## Projects

| ID | Name | Concepts | Difficulty |
|----|------|----------|------------|
| I01 | Port Scanner | TCP connections, threading, timeouts | Medium |
| I02 | Network Enumeration | DNS, ICMP, service detection | Medium |
| I03 | DNS Resolver | UDP sockets, protocol parsing | Medium |
| I04 | HTTP Server | tokio, async/await, routing | Medium |
| I05 | File Integrity Monitor | Hashing, file watching, database | Medium |
| I06 | Process Monitor | System calls, /proc parsing | Medium |
| I07 | Log Parser & Analyzer | Regex, statistics, reporting | Medium |
| I08 | Async Operations | tokio, futures, streams | Medium |
| I09 | Multi-threading | Arc, Mutex, channels | Medium |
| I10 | Database Operations | SQLite, rusqlite, migrations | Medium |
| I11 | REST API Client | reqwest, serde, authentication | Medium |
| I12 | WebSocket Client | tokio-tungstenite, real-time | Medium |
| I13 | System Info Collector | Cross-platform, sysinfo crate | Medium |
| I14 | Binary File Analysis | goblin, PE/ELF parsing | Medium-Hard |
| I15 | Memory Forensics Basics | Process memory, volatility | Hard |

## Skill Progression

```

    ↓
```
